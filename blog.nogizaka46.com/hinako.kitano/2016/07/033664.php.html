2016/0731Sun北野日奈子ラベンダーの香り

 
 
 
 
 
 
 
 
みなさまこんばんは！
 
 
 
北野日奈子です(｡・・｡)♪
 
 
 
 
今日はアルバム特典会の
ボーリング大会がありました～♪
 
 
 
 
ボーリングは苦手です( ˘ｰωｰ˘ )
 
チームの皆さんの足を引っ張ってしまいました( ˘ｰωｰ˘ )
 
この日のための練習しなかったからかな( ˘ｰωｰ˘ )
 
ほんと日奈子へたっぴでした( ˘ｰωｰ˘ )
 
 
ごめんなさい( ˘ｰωｰ˘ )
 
 
 
 
 
笑
 
 
 
 
でも、すごく楽しかったです♪
 
 
ありがとうございました！
 
 
 
 
 
 
 
 
いおりちゃん
かわいい
 
 
 
 
 
 
 
 
 
 
お団子こうげき～
 
 
 
 
 
 
 
 
 
 
 
昨日ね、家族みんなで夜ご飯を食べに行って
そこで初めてお酒のんでみました！！
 
 
 
ファジーネーブル
レモンチェロ ミルク割り
梅酒 ソーダ割り
 
 
をのんでみました！！
この中で1番好きなのは
ファジーネーブル！
 
 
甘くて美味しかったよ♪
 
 
梅酒のソーダ割りはまだ私には早いみたいです。。。
 
 
 
お父さんとお兄ちゃんが
すぐ赤くなっちゃうんだけど
私とお母さんもなんともなんなくて
北野家は女の人が強いことがわかりました！笑
 
 
 
妹が成人した時も初めての一口目は
家族みんなで集まれたらいいな♪
 
 
 
妹も強かったら
北野家女性陣つよいねっ(*´꒳`*)♪笑
 
 
 
 
 
 
お誕生日プレゼントにみさ先輩から
頂いたお酒も今度のもう♪
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
みさせんぱい
 
 
だーいすき！！！
 
 
 
 
 
 
 
 
 
 
 
 
あしたから8月ですね～
 
 
 
はやいですね～
 
 
 
時が経つのがはやい！！！
 
 
 
 
8月になっちゃうのに
お誕生日のことまだブログに書けてない！
 
 
 
やばいですね！書きます！
そろそろちゃんとまとめます！
 
 
 
たくさんケーキの写真載せなきゃ！
 
 
 
それではきょうはこのへんで！
 
 
 
おやすみなさい！
